# Pemrograman Berorientasi Objek

Repository ini berisi **hasil praktikum mata kuliah Pemrograman Berorientasi Objek**  
Program Studi **D4 Teknologi Rekayasa Komputer**  
**Politeknik Negeri Semarang**

Seluruh praktikum ditulis menggunakan **bahasa Python** dan dikerjakan melalui **Google Colab**.

---

## Deskripsi Repository

Repository ini digunakan sebagai dokumentasi dan arsip tugas praktikum Pemrograman Berorientasi Objek.  
Setiap file berisi latihan dan implementasi konsep algoritma dasar yang dipelajari selama perkuliahan.

Format file menggunakan **Jupyter Notebook (`.ipynb`)** agar mudah dijalankan dan dipahami secara bertahap.

---

## Teknologi yang Digunakan
- **Bahasa Pemrograman**: Python
- **Platform**: Google Colab
- **Format File**: Jupyter Notebook (`.ipynb`)

---

## Cakupan Materi Praktikum

Materi yang dipelajari dan diimplementasikan dalam repository ini meliputi:

- Dasar Bahasa Pemrograman Python
- Kelas dan Fungsi pada Bahasa Pemrograman Python
- Konstruktor, Destruktor, dan Enkapsulasi
- Inheritance
- Overriding

> Catatan: Materi pada setiap file menyesuaikan dengan silabus dan modul praktikum yang diberikan dosen.

---

## Cara Menjalankan File

1. Buka salah satu file `.ipynb` di repository ini
2. Klik **Open in Colab** atau upload file ke Google Colab
3. Jalankan cell satu per satu sesuai urutan

- Menjadi referensi pembelajaran Python dasar
